# Microservices Reading List

A curated list of the most popular and highly recommended blog posts, articles, and series that teach microservices architecture. These consistently rank at the top across developer searches, Reddit (/r/microservices, /r/softwarearchitecture), Hacker News, Medium, and recommendation lists.

---

## Foundational / Theory (Must-Read First)

These define what microservices are, their benefits, drawbacks, and core principles.

1. **"Microservices" by Martin Fowler & James Lewis (2014)**
   The original article that popularized the term. Explains the architectural style, characteristics (independent deployment, decentralized data, etc.), when to use it, and common pitfalls.
   **Link**: https://martinfowler.com/articles/microservices.html
   *(The #1 most-referenced introductory post of all time.)*

2. **NGINX 7-Part Series: "Microservices: From Design to Deployment" by Chris Richardson (2015)**
   A complete practical guide covering introduction, API Gateway, inter-process communication, service discovery, event-driven data management, and deployment. A compiled free ebook is also available.
   **Start here**: https://www.f5.com/company/blog/nginx/introduction-to-microservices
   *(Still one of the most recommended series for real-world design.)*

---

## Architecture Patterns & Best Practices

3. **microservices.io Patterns & Articles by Chris Richardson**
   The definitive catalog of 40+ patterns (Saga, Circuit Breaker, API Composition, Database per Service, CQRS, Outbox, etc.) with diagrams, pros/cons, and examples. Includes articles on decomposition, refactoring monoliths, and more.
   - **Patterns**: https://microservices.io/patterns/microservices.html
   - **Articles**: https://microservices.io/articles/
   *(The "bible" for microservices design patterns.)*

---

## Hands-On / Practical Tutorial Series

4. **JavaGuides / Ramesh Fadatare - 50+ Microservices Tutorials (Spring Boot + Spring Cloud)**
   One of the largest collections: building real microservices with Eureka, API Gateway, Docker, Kubernetes, design patterns, Saga, Circuit Breaker, and more.
   **Link**: https://medium.com/javaguides/my-50-must-read-microservices-tutorial-articles-and-guides-on-medium-platform-3e54a63b2671
   *(Hugely popular among Java/Spring developers.)*

5. **Baeldung Spring Microservices Guide**
   In-depth, code-heavy tutorials on Spring Cloud, Netflix OSS tools, Docker, etc.
   **Link**: https://www.baeldung.com/library/spring-microservices-guide

6. **"What are Microservices? Code Examples, Tutorials & More" by Stackify**
   Clear overview with practical code examples (Java, .NET) and best practices.
   **Link**: https://stackify.com/what-are-microservices/

7. **"Microservices Architecture Tutorial: All You Need to Get Started" (Educative)**
   Modern, concise intro covering benefits/drawbacks, Docker integration, patterns, and next steps.
   **Link**: https://dev.to/educative/microservices-architecture-tutorial-all-you-need-to-get-started-232g

## Recommended Learning Path

1. Fowler's article (theory foundation)
2. Richardson's NGINX series + microservices.io patterns (theory + design)
3. JavaGuides or your stack's tutorials (hands-on implementation)
